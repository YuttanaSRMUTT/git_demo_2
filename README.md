# gitdemo

```
$ echo "# git_demo" >> README.md
$ git init
$ git add README.md
$ git commit -m "first commit"
$ git branch -M main
$ git remote add origin https://github.com/YuttanaSRMUTT/git_demo_2.git
$ git push -u origin main
```
```
$ git add .
$ git status
$ git commit -m "2nd" OR git commit -a -m "2nd"
$ git push 
```

```
$ commit -a -m "2nd"
$ git push 
```
